Este proyecto consiste en la creación de un sitio web de e-commerce dinámico e interactivo, que consuma datos de una API REST para mostrar productos, y permita a los usuarios añadir productos a un carrito de compras. Tu proyecto deberá demostrar una sólida comprensión de la estructuración semántica, el diseño responsivo, la manipulación del DOM, la interacción con APIs, y la implementación de funcionalidades específicas de un e-commerce.
